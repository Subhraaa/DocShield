# DocShield - Secure Document Portal


A robust and secure web-based document portal designed for role-based access control, comprehensive audit logging, and secure document management.

## Features

### User Management
- **Registration**: Users can create new accounts securely.
- **Account Verification**: Implements a verification process to ensure user authenticity.
- **Login**: Secure authentication mechanism for authorized access.
- **Password Management**: Users can change their passwords to maintain account security.
- **Account Lockout**: Automatic account locking mechanism triggered after repeated failed login attempts to prevent brute-force attacks.

### Access Control & Permissions
- **Role-Based Access Control (RBAC)**: Users access resources and functionalities strictly according to their assigned roles.
- **Document Permissions**: Granular permissions allowing users to create or view documents based on their specific access rights.

### Security & Session Management
- **Token Refresh**: Mechanism to refresh expired access tokens, ensuring continuous and secure sessions without requiring re-login.
- **Audit History**: Comprehensive logging of security events, allowing users and administrators to view a detailed history of actions and access patterns.
